## ftool-gui

## 简介

在 macOS 上进行 frida 相关操作的图形化小工具

## 开发

```

git clone https://github.com/soft98-top/ftool-gui.git

cd ftool-gui

python -m pip install -r requirements.txt

pyinstaller --onefile --windowed --add-data "code:code" ftool-gui.py

```
